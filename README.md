# Sholpy

### E-commerse application

## Project Features
 - RESTFul API (Retrofit2)
 - Firebase Auth
 - Cloud Firestore 
 - Data binding 
 - View binding
 - Navigation component
 - Dark/Light mode 
 - ItemTouchHelper and RecyclerView 
 - Bottom Sheet Dialog
 - ViewPager2 
 - Picasso.
